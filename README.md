# Portfolio Website

Welcome! This is my personal portfolio website, designed and built using **Vue 3** and **Vuetify**. It serves as a showcase of my projects, technical skills, and professional experience.

## 🚀 Overview

This project is a modern, responsive, and internationalized web application. It features a dark/light theme, multilingual support, and 3D graphics.

## 🛠️ Tech Stack

- **Framework:** [Vue 3](https://vuejs.org/) (Composition API)
- **UI Component Library:** [Vuetify](https://vuetifyjs.com/)
- **Build Tool:** [Vite](https://vite.dev/)
- **State Management:** [Pinia](https://pinia.vuejs.org/)
- **Routing:** [Vue Router](https://router.vuejs.org/)
- **Internationalization:** [Vue I18n](https://vue-i18n.intlify.dev/) (Supporting English and Italian)
- **3D Graphics:** [Three.js](https://threejs.org/)
- **Time/Date Handling:** [Moment.js](https://momentjs.com/)
- **Icons:** [Material Design Icons (MDI)](https://materialdesignicons.com/)

## 🏗️ Project Structure

The project follows a standard Vue 3 + Vite structure:
- `src/components/`: Reusable UI components.
- `src/locales/`: Translation files (JSON) for English and Italian.
- `src/router/`: Navigation and routing configuration.
- `src/stores/`: Pinia stores for global state management (e.g., theme, language).
- `src/App.vue`: Root component.
- `src/main.ts`: Entry point where Vue is initialized with Vuetify, Pinia, Router, and I18n.

## 🛠️ Development Setup

### Install Dependencies

```sh
npm install
```

### Run for Development

```sh
npm run dev
```

### Build for Production

```sh
npm run build
```

### Linting & Formatting

```sh
npm run lint
```
